# AI-Based Intrusion Detection System (Hybrid RF-LightGBM)

## Overview

This project presents a hybrid machine learning model combining Random Forest and LightGBM for intrusion detection in network traffic.

## Key Features

* High accuracy (99.13%)
* Low computational cost
* Scalable for real-time applications

## Dataset

CICIoT2023

## Results

* Accuracy: 99.13%
* Precision: 99.05%

* F1-score: 99.08%
